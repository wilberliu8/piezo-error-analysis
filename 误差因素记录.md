## 因素一：线性度（Linearity）✅ 已聊透

### 定义
传感器实际输出曲线与理想直线之间的最大偏差，占满量程输出（FSO）的百分比。

### 常用计算方法
- **BFSL（Best Fit Straight Line / 最小二乘法）**：使所有标定点到直线的偏差平方和最小
- **端点法**：连接最小力点和最大力点做直线

### 计算公式
```
线性度(%FSO) = |实际电荷 - BFSL直线电荷|_max / FSO × 100%
```
其中 FSO = BFSL直线在满量程处的值（或最大-最小输出的范围）

### 灵敏度（Sensitivity）的确定
- 敏感度 = BFSL的斜率，单位 pC/N
- 标定完成后在电荷放大器里输入此值，放大器据此反算力值

### 标定规范要求
- ISO 376 推荐：≥8个均匀分布的标定点，每个力值重复3次以上

### 分段标定
- 通过缩小标定量程可提升有效精度（线性度误差绝对值随量程缩小而减小）
- 但量程不能无限缩小，受两个边界限制：
  1. **标定标准可及性**：小力值参考标准难以获得足够精度
  2. **长时间常数漂移**：漂移为绝对量（N/s），在小量程中占比过大，影响标定精度

### 与客户沟通的措辞（参考）
> "石英晶体的线性度从理论上来说是非常好的。如果BFSL线性度是0.2%，那么在该量程内，因线性度引起的误差大概率不会显著大于0.2%×满量程。当然这只是参考值，有可能会稍微大一些，但通常不会显著超过。如果客户需要更大量程，可以增加量程段的标定，得到新的灵敏度和对应的误差百分比——这就是分段标定的意义。"

### 输入参数（用于后续误差分析小程序）
| 参数 | 类型 | 说明 | 单位 |
|------|------|------|------|
| 标称量程 | 数值 | 传感器满量程 | N |
| 使用量程 | 数值 | 实际使用的量程段 | N |
| BFSL线性度 | 百分比 | 校准证书上的线性度 | %FSO |
| 线性度误差（绝对值） | 数值 | 自动计算 = 线性度 × 使用量程 | N |

## 因素二：漂移（Drift） ✅ 已聊透

### 物理根因
真实世界中不存在无穷大的阻抗，传感器-线缆-放大器之间存在电荷泄露，导致输出随时间缓慢变化。

漂移的两个物理分量：
| 分量 | 来源 | 时间行为 |
|------|------|---------|
| 偏置电流漂移 | 运放输入偏置电流 Ib 在 Cf 上积分 | 近似线性增长 |
| 泄露衰减漂移 | 有限绝缘电阻 R 导致 Q 通过 RC 回路衰减 | 指数衰减（t<<τ 时近似线性） |

### 典型数值（高端系统）
- Kistler 5018/5080 + 电荷输出传感器：~0.05 pC/s（恒温稳定工况）
- 若灵敏度 1 pC/N → 等效漂移 0.05 N/s

### 漂移的累积
漂移是**时间积分量**，测试时间越长，漂移累积越大：
```
漂移量(N) = 漂移率(N/s) × 测试时间(s)
```

### 量程依赖
漂移是绝对值（N/s），不随量程缩放，因此：
- **大量程测量**：漂移可忽略（如100,000N测1小时，漂移180N=0.18%）
- **小量程测量**：漂移成为主要限制因素（如1N测10s，漂移0.5N=50%）

### 温度敏感性与补偿方法
- 0.05 pC/s 是恒温稳定工况下的典型值，温度变化时漂移会显著增大
- 漂移有方向性（正向或负向），不是单向累积
- **如果漂移曲线肉眼可见是线性的** → 可以通过软件加反斜率曲线进行补偿
- **如果漂移曲线是非线性的** → 无法直接补偿，需要重复试验，等待温度稳定后再看结果

### 输入参数（用于后续误差分析小程序）
| 参数 | 类型 | 说明 | 单位 |
|------|------|------|------|
| 漂移率 | 数值 | 系统等效漂移率（可从校准证书或放大器规格获取） | pC/s |
| 灵敏度 | 数值 | 传感器的电荷灵敏度 | pC/N |
| 等效漂移率（自动计算） | 数值 | = 漂移率 ÷ 灵敏度 | N/s |
| 测试时长 | 数值 | 单次测量的持续时间 | s |
| 漂移误差（绝对值） | 数值 | 自动计算 = 等效漂移率 × 测试时长 | N |
| 测量目标值 | 数值 | 被测力的典型大小 | N |
| 漂移相对误差（自动计算） | 百分比 | = 漂移误差 ÷ 测量目标值 × 100% | % |

## 因素三：串扰（Crosstalk） ✅ 已聊透

### 定义
当施加纯单方向力时，其他方向通道出现的虚假输出信号。串扰是**绝对量**，与被测力和串扰率成正比。

### 串扰的三个来源

| # | 串扰源 | 层级 | 物理根因 | 主导场景 |
|---|--------|------|---------|---------|
| ① | **安装不平 / 几何耦合** | 晶体层面 | 晶体切割面与机械轴存在微小角度偏差，Z向力在X/Y方向产生几何投影分量 | 小力时主导（弹性变形小，几何因素是主因） |
| ② | **Poisson效应 / 侧向膨胀** | 晶体层面 | Z向晶体被压缩→泊松效应导致侧向膨胀→在X/Y剪切敏感晶环界面产生剪切应力 | 大力时主导（变形大，Poisson效应显著） |
| ③ | **偏载效应（载荷重新分布）** | 系统层面 | 测力台由4个传感器组成，偏载→力矩→各传感器载荷重新分布，因制造公差导致各传感器灵敏度不完全一致，求和后出现残差 | 实际切削/加载中几乎一直存在（力很少在正中心） |

### 规格表示方式（重要：区别于线性度）
- **线性度**用 **%FSO**（占满量程百分比，是绝对值概念：固定误差量不随加载力变化）
- **串扰（Cross talk）**用 **%实际力**（占被测力的百分比，是比例概念：加载力越大串扰量越大）

### 计算示例
```
Fz = 100N（实际加载）, cross talk = 1% Z→X, Z→Y
→ Fx(虚假) = 100N × 1% = 1N
→ Fy(虚假) = 100N × 1% = 1N
```
串扰误差与Fz成正比，物理上也合理（几何投影/Poisson效应/偏载效应都随加载力增大而增大）。

### 行业指标
| 产品 | 串扰规格（%实际力） |
|------|---------|
| Kistler 三向力传感器（单个） | ≤±2% |
| Kistler 9257BA 测力台 | ≤±1~2% |
| Kistler 9139AA 高端测力台 | ≤±0.5% |

### 补偿方式
通过**多分量标定**获得解耦矩阵（6×6或3×3矩阵），用软件算法抵消串扰分量。

### 输入参数（用于后续误差分析小程序）
| 参数 | 类型 | 说明 | 单位 |
|------|------|------|------|
| 被测力Fz | 数值 | Z向加载力 | N |
| 串扰率XY←Z | 百分比 | Z→X/Y的串扰率（%实际力） | % |
| 串扰误差X（自动计算） | 数值 | = 串扰率 × Fz | N |
| 串扰误差Y（自动计算） | 数值 | = 串扰率 × Fz | N |

## 因素四：外部环境噪声（振动噪声 + 声学噪声） ✅ 已聊透

### 为什么跳过电噪声

压电测力系统中，电荷放大器本底噪声典型值 0.01~0.02 pC RMS（窄带宽）。以灵敏度 1 pC/N 计，等效噪声力约 0.01~0.02 N。在非隔振环境下，环境振动通过 F=ma 耦合产生的虚假力通常是电噪声的 10~1000 倍。因此实战中电噪声占比极低，不作重点展开。

### 物理根因：F = ma 惯性耦合

```
F_vib = m_eq × a_env
```

环境振动加速度 a_env 作用在传感器安装面以上的等效质量 m_eq 上，产生惯性力，叠加到力传感器输出中。这个力不是被测对象产生的。

### m_eq 的构成

```
m_eq = m_传感器顶板 + m_夹具/适配器 + m_被测物体
```

传感器顶板本身几百克，夹具也几百克，合计通常 1~2 kg 起步。即使被测物体只有 1g，实际耦合质量可能是 2kg——虚假力往往比直觉估计大得多。

### 环境振动量级

| 场景 | 典型加速度 | 等效虚假力 (m_eq=2kg) |
|------|-----------|----------------------|
| VC-E 隔振实验室 | ~3 μg | ~0.06 mN |
| VC-A 普通实验室 | ~50 μg (0.05 mg) | ~1 mN |
| 厂房（远离路） | ~0.3 mg | ~6 mN |
| 厂房（30m外重卡经过） | ~1~3 mg | ~20~60 mN |

> **经验法则**：1 mg 振动 + 1 kg 等效质量 = ~10 mN 虚假力（不是 1 mN，注意 g=9.81）。

### 声学噪声——第二路径

声波通过两条路径产生虚假力：

| 路径 | 机制 | 示意计算 |
|------|------|---------|
| ① 直接压力 | 声波压在物体表面，F = p × A | 80 dB (0.2 Pa) × 0.01 m² = 2 mN |
| ② 声-结构共振 | 声波激励安装台架弯曲模态，振动经底座传至传感器，Q 值放大 10~50 倍 | 2 mN × Q(20) = 40 mN |

对精密力测量，声学噪声可与地面振动同量级。

### 量程依赖（与漂移同病）

振动虚假力是**绝对值（N）**，不随被测力缩放：

| 场景 | 被测力 | 虚假力 (2kg, 1mg) | 误差占比 | 应对 |
|------|--------|-------------------|---------|------|
| 大量程（切削力） | 100 N | ~20 mN | 0.02% | 可忽略 |
| 小量程（微装配力） | 0.1 N | ~20 mN | 20% | 致命 |
| 精密流体力学 | ~0.01 N | ~20 mN | 200% | 必须解决 |

**结论：小力测量时，环境振动是真正的精度天花板。**

---

### 应对方案（分层）

#### 第一层：物理隔绝（无副作用，优先级最高）

| 手段 | 针对 | 原理 | 效果 |
|------|------|------|------|
| **隔振台**（气浮/主动） | 地面振动 | 低刚度弹簧解耦台面与地面 | 10Hz 以上衰减 20~40dB（99%~99.99%）；固有频率以下（1~3Hz）不衰减 |
| **隔声罩** | 空气声波 | 密实罩体反射/吸收声波 | 降噪 20~40 dB |
| **轻量化夹具 + 刚性固定** | 残余振动 | 降低 m_eq，减小惯性力 | m_eq 从 2kg 降到 0.5kg → 虚假力降 4 倍 |

#### 第二层：信号补偿

##### 方案 A：加速度补偿

在传感器旁安装参考加速度计，实时测量 a(t)，软件减去 m_eq × a(t)。

- **瓶颈**：m_eq 不是常数，随频率变化（不同频率参与振动的模态质量不同）。保守估计 m_eq 标定误差 2~5%。
- **对 0.1mN 目标**：20 mN × 2% = 0.4 mN 残差 → 困难
- **成本**：一个加速度计（~几千元）

##### 方案 B：时间差分统计平均（★ 推荐，零额外成本）

**核心思想**：利用"被测信号确定 + 噪声随机"的统计特性，通过多次实验平均提升信噪比。

**原理类比**：声学主动降噪（ANC）。不需要把噪声频率成分降到零，只需要把"干扰部分"从总信号中扣除大部分即可。即便被测信号和噪声有相同的频率成分（如 2~5Hz），也不影响——关键是信号的确定性 vs 噪声的随机性，不是频率分离。

**操作流程**：

```
阶段一：对照组（关阀/空载）
  → 采集多组纯噪声，建立噪声频率特征库
  · 主频列表（隔振后通常只有 2~3 个低频峰）
  · 各频率幅度的统计量（均值、方差）
  · 确认时间稳定性

阶段二：实验组（开阀/加载）
  → 相同工况重复 N 次
  → 自动获得：
     · 信号 S（确定性，每次相同）
     · 噪声 Nᵢ（随机相位，每次不同）
  → N 次复数平均：
     avg(S + Nᵢ) = S + avg(Nᵢ)
     噪声衰减 ~1/√N
```

**为什么不需要相位对齐**：

单次实验的频域复数平均本身就是天然滤波器。噪声相位在每次实验中随机分布 → 复数叠加时互相抵消。信号相位每次都一样 → 不衰减。

| 实验次数 N | 噪声衰减 | 举例：20mN 噪声→ |
|-----------|---------|-----------------|
| 10 | 1/√10 ≈ 32% | → 6.3 mN |
| 100 | 1/√100 = 10% | → 2 mN |
| 400 | 1/√400 = 5% | → 1 mN |
| 2500 | 1/√2500 = 2% | → 0.4 mN |

**优势**：
- 零额外硬件成本
- 不需要新增传感器或改变工装
- 对照组同时覆盖振动和声学噪声（不需要区分来源）

**注意事项**：
- 拉开实验间隔（相邻实验间等 5~10 秒），破坏建筑低频振动的相位连续性，保证 Nᵢ 真正随机
- 实验前通过对照组估算噪声量级 → 反推需要多少次实验才能达标

##### 方案 C：差动补偿

两套传感器+质量，一套测力一套纯参考，取差值。

- **流体力测量中的位置悖论**（不可解）：
  - 参考传感器放隔振台外面 → 振动量和台面不匹配，不是共模
  - 参考传感器放隔振台上面 → 管路流致振动也传到参考传感器，被污染
- **结论**：在管路流体力测量场景中，差动补偿因物理位置悖论而不可行

---

### 关键结论

1. **物理隔绝（隔振台+隔声罩+轻量化）是最可靠的第一道防线**，没有副作用
2. **时间差分统计平均是性价比最高的信号补偿方案**，零额外硬件，理论基础坚实
3. 信号补偿只能打辅助——先把物理隔绝做到位，补偿才有意义
4. 低频段（<5Hz）即使有信号也无妨，因为靠的是统计平均而非频段分离

## 因素五：频响误差（高频机械共振） ✅ 已聊透

### 范围说明

频响误差包含两个方向：
- **低频下限**：电荷放大器 RC 时间常数。很简单：$f_{low} \approx 1/(2\pi\tau)$，τ 越大低频越好。不展开。
- **高频上限**：传感器+安装+工件的机械共振。这是本因素的焦点。

### 物理模型

压电测力传感器是二阶质量-弹簧系统，频响幅值：

$$|H(r)| = \frac{1}{(1-r^2)}, \quad r = f/f_n \quad (\zeta \approx 0)$$

> 阻尼极小（ζ ≈ 0.01~0.03），忽略不计，公式已足够准确。

### 经验法则（高频）

| 频率比 r = f / f_n' | 近似误差 | 适用场景 |
|---------------------|---------|---------|
| ≤ 1/5 | ~5% | **精确定量测力** |
| ≤ 1/3 | ~10~12% | **一般定量测力** |
| 1/2 | ~33%（~2.5dB） | **频率成分识别 / 特征检测**（增益让特征峰更突出，反而有利） |
| > 1/2 | 急剧增大 | 一般不用于定量 |

> 这些都是工程经验值，大约判断用的。实际值在这个范围附近波动，给客户快速估算用。

### 附加质量修正

$$f_n' = f_n \times \sqrt{\frac{m_s}{m_s + M}}$$

其中：
- $f_n$：传感器标称固有频率（手册值）
- $m_s$：传感器参与振动的等效质量（≈ 顶板质量，几十到几百克）
- $M$：附加质量（夹具 + 工件）

**示例**：$m_s = 1$, $M = 8$, $f_n = 30\text{ kHz}$ → $f_n' = 30 \times \sqrt{1/9} = 10\text{ kHz}$。5% 带宽 ≈ 2 kHz。

### 实战注意事项

1. **手册 $f_n$ 不可直接用**：厂家标称的固有频率是在规定安装条件（钢块、规定扭矩）下测的。实际安装（夹具、底板刚度、安装面平整度）会显著拉低 $f_n$。有条件的建议装后实测。
2. **$m_s$ 不是传感器总质量**：只有传感器顶板以上的等效质量参与振动，通常远小于传感器总重。
3. **场景决定误差容忍度**：定量测力选 1/5；特征识别到 1/2 都可以用，增益反而是好事。见上表第三行。
